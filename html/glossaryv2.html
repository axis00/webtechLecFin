Glossary of Terms

B

Broken Authentication

An attack caused by bad implementation of the authentication and/or session management of a web application is considered as a broken authentication.

C

Cross-Site Scripting

Cross-Site Scripting or XSS is an attack that attacks the website users instead of targetting the website. It is similar to Injection attacks. However, this occurs when an attacker embed scripts that would run on a victim's browser without the victims knowledge. This can occur often since modern browsers are capable of running scripts.

E

Expression Language

Expression language allows access to different components of JavaBeans.

I

Injection

An injection attack happens when a malicious input is entered into a web application. This input can be a command, query, or malware. Once this input is injected in the program, the interpreter may change the behavior of the program.

J

JavaScript

JavaScript is a dynamic programming language that can be applied to an HTML file which provides different interactivities to the websites.

JavaServer Page

JavaServer Pages (JSP) is a server-side programming technology that allows the developer to create dynamic web applications.

Java Servlet

Java Servlets are programs that handle HTTP requests and generate HTTP responses. This technology is hosted in a servlet container that provides the environment in which the servlet runs, and it also serves as a means of controlling the servlet.

JSTL

JSP Standard Tag Library are collection of tags used in a JSP file. These tags creates a container for Java codes, and separates it from the other components of a JSP file.

N

National Privacy Commission

National Privacy Commission or NPC is an independent body that takes charge in administering and implementing the Data Privacy Act of 2012 in the Philippines.

P

PHP

Hypertext Preprocessor (PHP) is an open source scripting language that is primarily used for web applications.

S

Scriptlets

Scriptlets are Java codes embedded in JSP pages.

W

Web Applications

Web applications are programs from a server that can be run and used in multiple computers through web browsers.

Web Security Threats

Web security threats are malicious activities that could possibly attack any application connected to the web.
